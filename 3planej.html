Planejamento Estratégico de TI

 

Documentar os requisitos para o Simulador da Casa Bancária. Construir o modelo de documento de análise (MDA) : requisitos, diagrama de casos de uso (com descrição de cada caso de uso contendo, pelo menos, protótipo da tela e fluxos envolvidos em cada tela), diagrama de classes (com descrição de cada classe, obedecendo o padrão BCE para as classes - 16 Border, Control e Entity - um diagrama de classe para cada caso de uso), diagrama de sequência (para cada diagrama de classe) e o diagrama de atividades (descrevendo as interações entre os casos de uso).
